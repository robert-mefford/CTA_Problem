# CTA_Problem

### To Start:
node code.js

### Input:
Input number array.
The First value is fare amount. And the other values are Gavvo Pieces.

### Output:
If there's possible combination of Gavvoes then print it using array.
If not, print "not possible"

#### Input: 7 1 4 6 2 3
#### Output: [1, 6]

## Example2:

#### Input: 57 6 13 21 23 24
#### Output: [23, 21, 13]

## Example3:

#### Input: 8 1 4 6
#### Output: not possible
